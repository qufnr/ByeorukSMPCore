## 동물농장 정상화
- 각 동물에 상태 부여 (상태: 마름, 보통, 통통함)
  * 자연 생성된 동물은 "보통" 상태
  * 먹이를 많이 줄 수록 상태 좋아짐
- "마름"상태에서 도축 시 드롭률 0%
- "보통"상태에서 도축 시 드롭률 20%
- "통통함"상태에서 도축 시 드롭률 50%
- 교배는 "통통함" 상태인 동물끼리 자동으로 교배하며, 각 동물에게는 교배할 수 있는 횟수가 정해짐
- "마름" 상태에서 먹이를 주지 않으면 죽어버림
- 동물을 바라보고 있으면 동물 위에 상태가 표시되며, 동물에게 이름이 부여되어 있을 경우 이름이 맨 위에 표시되고 아래에 상태가 표시 됨
